FAN FIGHT CARDS

Deiveson Figueiredo vs Alex Perez

Deiveson is the current number #1 ranked contender in the Flyweight division, and has been unstoppable in this division with only one loss in eight Octagon appearances, the only loss coming off against Jussier Formiga in 2019. Alex Perez, the current number #4 flyweight contender, also has had a terrific run in this division, with his last win commig off against Formiga. This would be a great fight, possibly even a great title fight, to lighten up this division.
